      An anomaly, or outlier, is simply defined as something that does not conform to the
      expected also known as outlier detection, is about identifying those observations that
      are anomalous. Time series data are observations over a period of time. In this case your
      dataset has a time stamp. With time series, an anomaly detection algorithm will based on
      historical data identify observations that does not conform to the expected. Historical data consist of nominal and anomalous data point

      Methods for detect Anomaly

      1. Supervised learning we consider it as supervised learning probelm if following condition met

        a). When we have labelled training data b). Anomalous and normal classes are balanced ( say at least 1:5)
        What is Anomaly Actually

      Anomalies or outliers come in three types.

      1. Point Anomalies. If an individual data instance can be considered as anomalous with respect to the rest of the data

      2. Contextual Anomalies, If a data instance is anomalous in a specific context, but not otherwise.

      3. Collective Anomalies. If a collection of related data instances is anomalous with respect to the entire dataset, but not individual values. They have two variations.
        Events in unexpected order
        Unexpected value combinations

      Solution
      probability of a sequence of events happening. the probability of that sequence occurring and use that to detect any rare sequences.
      P(A|B) = P(A & B)/P(B)
